On-premises Data Plane

The on-premises data plane is used to monitor applications deployed in the on-premises environment.

The supported products in the on-premises data plane are TIBCO ActiveMatrix BusinessWorks™ 5 and TIBCO ActiveMatrix® Adapters.

The supported adapters are as follows:

  • TIBCO ActiveMatrix® Adapter for Database

  • TIBCO ActiveMatrix® Adapter for Files for Unix/Win

  • TIBCO ActiveMatrix® Adapter for LDAP

  • TIBCO ActiveMatrix® Adapter for SAP

For more information, refer Products documentation.
